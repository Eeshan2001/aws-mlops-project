# MLOps Topic Modeling on AWS

## Overview

This project demonstrates a complete MLOps pipeline for deploying a Topic Modeling application on AWS using:

- Flask
- Gunicorn
- Docker
- Amazon ECS
- Amazon ECR
- AWS CodePipeline
- AWS CodeBuild
- Amazon S3
- Terraform

The application predicts topics from user-provided text using an LDA model and deploys automatically through a CI/CD pipeline.
